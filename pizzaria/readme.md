Implementa API de pizzaria:
-CRUD de clientes;
-CRUD de pizzas;
-Criar pedido;

## Especificações
- PHP 7.4
- Composer

## Execução
1. `composer install`
2. `php -S localhost:8000 -t public`

## Testes
`composer run test`
